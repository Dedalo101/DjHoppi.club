# DjHoppi.club

Static site for DjHoppi.club, inspired by lovefaders.com, amoro.club, and bennyasoto.com, with a craft beer theme.

## Features

- Generative visuals: hops plants and beer bubbles (green to black theme)
- Social media links: Mixcloud, Soundcloud
- Section for local brew company advertisements
- Responsive, modern design

\n## How to preview
Open `index.html` in your browser. All assets are local and require no build step.

## Deployment

To deploy on GitHub Pages, push these files to your repository and enable Pages in repository settings.

## Customization

- Replace the placeholder in the ads section with actual brewery ads.
- Adjust visuals in `visuals.js` for more complex shaders or effects.
